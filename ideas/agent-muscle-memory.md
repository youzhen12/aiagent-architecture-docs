# 关于 Agent 的一些想法

一个 agent 就像一个人，根据输入输出相应的动作。比如我要在电脑上打字，那我要控制我的手去做这件事情。大模型现在已经基本具备了这样的智能，它不会去让耳朵去干这件事情。

多 agent 协作就好像我们写方案、写代码，需要人去评审。一个人对于自己完成事情会有先入为主的幻觉，觉得自己写的是正确的；大模型也一样，会产生这样的幻觉。这时候就需要另外一个人、一个 agent 去做监督对抗，提升正确性。

另外一个观点：人有大脑记忆，agent 现在也有记忆模块。人还有肌肉记忆，但是现在关于 agent 的「肌肉记忆」很少被提及。我的想法是，在确定用户的想法后，可以提供一些按钮用于固定的动作，就不需要每次都去问大模型再调工具，这样在一些场景下可以明显提升效率。
