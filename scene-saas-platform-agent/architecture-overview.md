# 1. 场景背景与目标

- 业务背景：
  - 多租户 SaaS 平台，包含控制台、API、计费、权限、配置、运维等模块。
  - 用户角色多样：管理员、运营、开发者、财务、合作伙伴等。
- AIAgent 目标：
  - 提供一站式「产品顾问 + 配置助手 + 运维助手」，降低使用门槛。
  - 帮助用户理解功能、快速完成配置、排查常见问题。
  - 协助平台团队收集需求与反馈，优化产品体验。

# 2. 多角色与多租户设计

- 多租户约束：
  - 所有 Agent 与工具调用必须在租户隔离边界内执行。
  - Prompt 中显式包含租户 ID、环境（prod/stage）、角色（admin/dev 等）。
- 典型角色：
  - `AdminAgent`：面向管理员，指导权限配置、策略设置、审计查看等。
  - `DevAgent`：面向开发者，聚焦 API 集成、SDK 使用、调试排错。
  - `OpsAgent`：面向运维/值班人员，围绕监控、告警、扩容、版本发布等提供建议。

# 3. 能力边界与责任划分

- AIAgent 能做的：
  - 生成基于当前产品与文档的操作指引与配置建议。
  - 调用平台 API 执行「可自动化且可回滚」的操作（如创建测试环境配置）。
  - 帮助用户理解告警信息、资源使用情况与账单结构。
- AIAgent 不应做的：
  - 直接执行高风险、不可逆操作（如删除生产资源、停用关键服务）。
  - 在未获得显式确认前修改计费相关参数或安全策略。

# 4. 总体技术架构

- 推荐架构：
  - 控制台前端 / CLI / Chat 插件 -> AI 网关 -> Orchestrator（多 Agent 编排） -> 工具层（SaaS 内部 API、监控、计费、权限服务）。
- Orchestrator 职责：
  - 识别当前场景（配置指导、问题排查、使用咨询、账单问题等）。
  - 对接不同角色 Agent，并在需要时启动多步工作流（例如「分析告警 -> 生成排查步骤 -> 给出自动化脚本草稿」）。

# 5. 模型、向量库与配置选型

- 模型选型：
  - 支持长上下文，方便一次性加载多份产品文档、API 参考与配置示例。
  - 对结构化信息理解能力强（JSON、YAML、日志、监控数据等）。
- 向量库与文档：
  - 切分维度：按产品模块（权限、计费、API、监控）、版本（v1/v2）、环境（公有云/专有云）组织文档。
  - 对于高频配置示例，保留结构化模板，结合 RAG 提供「示例 + 解释」。

# 6. 工具设计与安全策略

- 工具分类：
  - 只读工具：查询租户配置、资源状态、告警、账单、审计日志。
  - 变更类工具：创建/更新配置、开通/关闭功能开关、触发发布任务等。
- 安全控制：
  - 所有变更类工具必须要求显式确认字段（如 `confirm: true`），并在前端进行二次确认。
  - 变更操作应默认在「沙盒/测试环境」执行，生产环境需人工审核或更高权限。
  - 对于 CLI/Terraform 脚本生成，Agent 只返回建议脚本，不直接执行。

# 7. 典型用户旅程示例

- 新租户启动：
  1. 用户描述业务目标，`AdminAgent` 基于模板推荐基础配置方案。
  2. Agent 生成权限角色、组织结构与初始策略草案，供管理员审核一键应用。
- API 集成开发：
  1. 开发者粘贴现有代码或错误日志，`DevAgent` 帮助定位签名、权限或参数问题。
  2. 根据 SDK 文档与示例自动生成调用样例与错误处理逻辑。
- 告警排查：
  1. `OpsAgent` 读取告警详情、相关监控指标，与近期变更记录联合分析。
  2. 返回「可能原因 + 影响范围 + 建议操作步骤」，并可生成自动化脚本草稿。

# 8. 监控、审计与计量

- 监控与审计：
  - 对 AI 相关操作单独打标签（如 `source=ai_agent`），便于区分人工与 AI 行为。
  - 记录每次工具调用的请求参数摘要与结果状态，用于问题回溯与责任界定。
- 成本与计量：
  - 统计各租户的 AI 使用情况（调用次数、Token、成功率等），便于后续计费与配额管理。
  - 可考虑提供「AI 帮手」作为增值功能，通过套餐/配额等形式售卖。

# 9. 实施路线建议

- 第一阶段：
  - 聚焦「文档问答 + 控制台引导」，不直接执行任何写操作。
  - 完成基础 RAG 能力与前端集成，收集用户问题数据。
- 第二阶段：
  - 引入只读工具与少量低风险写操作（测试环境配置、Demo 资源创建）。
  - 完善多角色 Agent 分工与 Orchestrator 策略。
- 第三阶段：
  - 构建完整的运维助手与账单助手，引入更完善的安全隔离、审批与审计机制。
